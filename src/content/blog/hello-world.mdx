# Hello World

SCLog에 오신 것을 환영합니다! 이 블로그는 Astro와 MDX를 사용하여 만들어졌습니다.

## Astro란?

Astro는 콘텐츠 중심의 웹사이트를 만들기 위한 프레임워크입니다. 빌드 타임에 HTML을 생성하여 빠른 로딩 속도를 제공합니다.

## MDX의 장점

MDX를 사용하면 Markdown 문서 안에서 JSX 컴포넌트를 사용할 수 있습니다.

### 코드 하이라이팅 예시

```typescript
interface BlogPost {
  title: string;
  pubDate: Date;
  description: string;
  tags: string[];
}

function formatDate(date: Date): string {
  return date.toLocaleDateString('ko-KR', {
    year: 'numeric',
    month: 'long',
    day: 'numeric',
  });
}
```
