# 들어가며: 이 글도 AI가 쓴 거임

먼저 고백하자면, **이 블로그 글도 AI가 썼다.**

Claude한테 "지금까지 블로그 만든 과정 갖고 글 써줘, AI 어그로 팍 끌거야"라고 했더니 진짜 써줌.

심지어 "조금 더 짜치게, 빙수 먹었다는 내용도 넣어줘"라고 했더니 그것도 반영함.

그러니까 지금 이 문장도... 맞음. AI임.

# 타임라인: 내가 뭘 했고 AI가 뭘 했나

## 오후 5시 - 시작

```
나: "블로그 만들어줘"
Claude: "넵. Astro로 할까요?"
나: "ㅇㅇ"
```

→ 프로젝트 생성됨

## 오후 5시 30분 - 빙수 타임

AI가 디자인 시스템 만드는 동안 나는 **팥빙수**를 먹음.

돌아와보니 라이트/다크 모드, 컬러 토큰, 타이포그래피 전부 세팅되어 있었음.

## 오후 6시 - 유튜브 타임

```
나: "읽기 경험 개선해줘. 목차, 코드 복사 버튼, 진행률 바 같은 거"
Claude: "스펙 작성할게요"
나: "ㅇㅋ"
```

→ AI가 스펙 쓰고 리서치하고 태스크 분해하는 동안 나는 유튜브 봄

→ 돌아와서 "고" 한 마디 함

→ 구현 완료됨

## 오후 7시 - 갑자기 욕심남

```
나: "프로필 사진에 마우스 올리면 멋있는 애니메이션 있으면 좋겠어"
Claude: (회전 애니메이션 구현)
나: "3D로 동전처럼 회전했으면 좋겠어!"
Claude: (perspective, rotateY로 수정)
나: "메인 머지행"
```

이게 진짜 대화 전문임. "3D로 해줘" 한 마디에 CSS 3D 트랜스폼 코드가 뚝딱.

# 증거: 실제로 AI가 짠 코드

## 3D 동전 뒤집기

내가 "3D로 동전처럼"이라고 했더니 나온 코드:

```css
.profile-avatar {
  perspective: 1000px;
}

.avatar-default,
.avatar-alternate {
  backface-visibility: hidden;
  transition: transform 0.6s ease;
}

.avatar-alternate {
  transform: rotateY(180deg);
}

.profile-avatar:hover .avatar-default {
  transform: rotateY(180deg);
}

.profile-avatar:hover .avatar-alternate {
  transform: rotateY(360deg);
}
```

`perspective`? `backface-visibility`? 나는 이런 거 시킨 적 없음. AI가 알아서 넣음.

## 접근성도 알아서

```css
@media (prefers-reduced-motion: reduce) {
  .avatar-default,
  .avatar-alternate {
    transition: opacity 0.1s ease;
  }
  /* 회전 빼고 페이드만 */
}
```

접근성 얘기 한 적 없는데?

프로젝트 헌법(Constitution)에 "접근성 중요"라고 써놨더니 자동 적용됨.

# Speckit: AI한테 일 시키는 꿀팁

그냥 "만들어줘"라고 하면 AI도 뭘 만들지 모름.

**Speckit**이라는 워크플로우 씀:

```
/speckit.specify 프로필 hover 애니메이션 추가해줘
```

이러면:
1. 스펙 문서 자동 생성
2. 기술 리서치 (어떤 방식이 좋을지)
3. 태스크 분해 (뭐부터 할지)
4. 구현

나는 중간에 "ㅇㅋ", "고", "메인 머지행"만 입력함.

# 솔직 후기

## 좋았던 점

- **빙수 먹을 시간 생김**: AI가 일하는 동안 나는 쉼
- **일관성**: 내가 짜면 파일마다 스타일 다른데, AI는 일관적
- **공부됨**: AI 코드 보면서 "아 이렇게도 하는구나" 배움

## 심심했던 점

솔직히 **기다리는 시간이 좀 심심함**.

AI가 스펙 쓰고 리서치하고 구현하는 동안 할 게 없음.

그래서 빙수 먹고 유튜브 봄.

## 아쉬웠던 점

- 가끔 context 날아가서 "아까 그거"가 안 통함
- "이거 해줘"는 잘 되는데 "이거 하지 마"는 어려움
- 복잡한 비즈니스 로직은 아직 사람이 봐야 함
